### README.md
Gerador de Currículo em PHP

Uma aplicação web profissional para gerar currículo com formulário interativo, campos dinâmicos e impressão em PDF.

Funcionalidades

-  Formulário completo com dados pessoais, formação e experiência
-  Campos dinâmicos para adicionar múltiplas experiências, formações e referências
-  Cálculo automático de idade
-  Validação de formulários
-  Geração de PDF via impressão
-  Interface responsiva com Bootstrap 5
-  Versionamento com Git

Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript, jQuery
- **Backend**: PHP 7.4+
- **Framework CSS**: Bootstrap 5
- **Servidor Local**: XAMPP (Apache + PHP)
- **Versionamento**: Git e GitHub

Requisitos

- XAMPP instalado (Apache, PHP, MySQL)
- VS Code ou outro editor de texto
- Git instalado
- Conta GitHub
